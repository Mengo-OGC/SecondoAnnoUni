Importante seguire le code convenzioni. I nomi di package vanno sempre con le minuscole. metodi e variabili in camelCase, classi in PascalCase, e costanti SNAKE_CASE.

buona cosa dividere sorgenti da compilati, quindi per dire a javac con il flag si può dire dove mettere i compilati con "-d path". buona cartella usare "bin".
anche in javac si usa -cp

con java si eseguono ".class" si usa "-cp path" per trovare i .class delle classi. -cp è il classpath il percorso delle classi che usano al loro interno la classe.
java -cp "directory" classpath
il classpath si specifica con i "." come separatori.
